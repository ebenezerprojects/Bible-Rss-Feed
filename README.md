# Bible API

Complete Bible API with authentication, multi-version support, RSS feeds, and format management.

## Features
- JWT Authentication
- Multi-version Bible support (KJV, Tamil)
- Per-user version access control
- RSS feeds (verse and reference)
- Multiple formats (XML, JSON, TXT)
- Request tracking and analytics
- Rate limiting
- Comprehensive logging

## Quick Start

```bash
# Install dependencies
npm install

# Initialize database
npm run init-db

# Seed sample data
npm run seed

# Start development server
npm run dev
